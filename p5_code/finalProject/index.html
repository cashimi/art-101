Final Project

You graduated college, and now it's time to move out of your parent's basement. However, this job is an absolute bore. Controls are best discovered during the simulation, though there is a reference below and within the fame. This simulation can only be ran on a Google Chrome browser and best experienced with sound and console log open.

CONTROLS

  • First Click: Start Simulation
  • 'T': Access Computer
  • 'C': Clear calculator input/output (only for calculaor screen)
  • 'R': Access "new" office
  • 'H': Help Screen
  • '1': Return to main desk (restarting)
